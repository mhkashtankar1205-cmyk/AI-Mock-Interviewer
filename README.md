Author(s): Mahak Ashtankar  
Affiliation: RTMNU

---

## 📄 Abstract
The AI Mock Interviewer System is designed to simulate real interview environments using artificial intelligence. It helps students and professionals practice interviews by providing role-based questions and real-time feedback on their responses. The system evaluates answers based on clarity, confidence, and relevance, enabling users to improve their communication and interview performance.

---

## 📌 Introduction
Preparing for interviews is a critical yet challenging task for students and job seekers. Traditional preparation methods lack real-time feedback and realistic simulation.  

This system provides an AI-powered interview experience where users can practice anytime and receive instant feedback, improving their readiness and confidence.

---

## 📚 Literature Review
Existing platforms like LeetCode and InterviewBit focus on coding but lack behavioral interview simulation. Some mock interview tools exist but do not provide real-time AI-based feedback.  

Thus, there is a need for an intelligent system that simulates interviews and evaluates performance dynamically.

---

## ⚙️ Methodology
- Input: User selects role  
- AI generates interview questions  
- Speech/Text input captured  
- AI analyzes responses  
- Feedback generated  

---

## 💻 Implementation
Frontend: React.js  
Backend: Node.js / Python  
AI: Claude API / NLP models  
Speech: Web Speech API  

---

## 📊 Results and Discussion

- Real-time feedback accuracy  
- Effective role-based questioning

## ⚠️ Limitations

- Dependency on speech accuracy
